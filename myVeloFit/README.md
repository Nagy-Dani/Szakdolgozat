# myVeloFit

A desktop bike-fitting application built with Python and PyQt6.

Upload a side-view video of yourself pedaling, enter your body measurements, and get actionable recommendations to optimize your bike fit.

## Quick Start

```bash
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
python main.py
```

## Architecture

This project follows the **MVC (Model-View-Controller)** pattern. See `PROMPT.md` for the full specification.
